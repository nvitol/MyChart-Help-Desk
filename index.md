# How to Sign Up for a MyChart Account With an Activation Link or Code

## Purpose

This document defines the approved steps for creating a MyChart account using either:
- A MyChart activation link, or
- A MyChart activation code

---

## Eligibility

- Patients **12 years of age or older** are eligible to create a MyChart account using this process.

---

## When to Use This Procedure

Use this procedure when:
- A patient has received a MyChart activation link by email or text message
- A patient has received a printed MyChart activation code
- Activation credentials were provided by patient-facing staff

---

## Activation Options

Patients may sign up using one of the following methods:
- **Activation link**
- **Activation code**

Follow the applicable procedure below.

---

## Procedure: Sign Up Using an Activation Link

### Step 1: Open Activation Link

1. Open the email or text message containing the MyChart activation link.
2. Select **Sign up for MyChart**.

The link opens the MyChart sign-up page. [1](https://unityhealth.to/wp-content/uploads/2024/11/mychart-sign-up-with-activation-code.pdf)

---

### Step 2: Create Login Credentials

1. Enter a MyChart username.
2. Enter a MyChart password.
3. Enter the patient’s **Date of Birth**.
4. Select **Submit**. [1](https://unityhealth.to/wp-content/uploads/2024/11/mychart-sign-up-with-activation-code.pdf)

---

### Step 3: Accept Terms and Conditions

1. Read the **MyChart Terms and Conditions**.
2. Select **Accept**.

After acceptance:
- The MyChart account is active.
- The user may begin using MyChart features. [1](https://unityhealth.to/wp-content/uploads/2024/11/mychart-sign-up-with-activation-code.pdf)

---

## Procedure: Sign Up Using an Activation Code

### Step 1: Access MyChart

1. Visit **https://mychart.unityhealth.to**, or
2. Open the MyChart app from the Apple App Store or Google Play Store.
3. Select **Sign up now**. [1](https://unityhealth.to/wp-content/uploads/2024/11/mychart-sign-up-with-activation-code.pdf)

---

### Step 2: Enter Activation Information

1. Enter the **MyChart activation code**.
2. Enter the patient’s **Date of Birth**.
3. Enter one of the following identifiers:
   - **OHIP number**, or
   - **Medical Record Number (MRN)**
4. Select **Next**. [1](https://unityhealth.to/wp-content/uploads/2024/11/mychart-sign-up-with-activation-code.pdf)

---

### Step 3: Set Up Two-Step Verification

1. Select **Continue** to enable Two-Step Verification.
2. Add an email address or phone number.
3. Select **Continue**.
4. Enter the verification code received by email or text message.
5. Select **Verify**. [1](https://unityhealth.to/wp-content/uploads/2024/11/mychart-sign-up-with-activation-code.pdf)

---

### Step 4: Create Login Credentials

1. Create a MyChart username.
2. Create a MyChart password.
3. Select **Next**. [1](https://unityhealth.to/wp-content/uploads/2024/11/mychart-sign-up-with-activation-code.pdf)

---

### Step 5: Set Communication Preferences

1. Configure communication preferences as prompted.
2. Select **Sign in**. [1](https://unityhealth.to/wp-content/uploads/2024/11/mychart-sign-up-with-activation-code.pdf)

---

### Step 6: Accept Terms and Conditions

1. Read the **MyChart Terms and Conditions**.
2. Select **Accept**.

After acceptance:
- The MyChart account setup is complete.
- The user may begin using MyChart. [1](https://unityhealth.to/wp-content/uploads/2024/11/mychart-sign-up-with-activation-code.pdf)

---

## Out of Scope

This document does not cover:
- Medical or clinical questions
- Login issues for existing MyChart accounts
- Password resets for established users
- Epic or MyChart system outages

# MyChart Two-Factor Authentication and Passkey

## Purpose

This document defines:
- What Two-Factor Authentication (2FA) is in MyChart
- How to complete Two-Factor Authentication
- What a passkey is
- How to use a passkey to sign in to MyChart

---

## Definitions

### Two-Factor Authentication (2FA)

Two-Factor Authentication is a security feature that requires:
1. A MyChart username and password
2. A verification code sent to a trusted email address or phone number

### Passkey

A passkey is a secure sign-in method that allows access to MyChart without entering a password.  
Passkeys use device-based security, such as:
- Fingerprint
- Face recognition
- Device PIN

---

## When Two-Factor Authentication Is Required

Two-Factor Authentication is required:
- During the first login after account creation
- When signing in from a new device
- When signing in from an unrecognized browser
- When security settings require additional verification

---

## Procedure: Complete Two-Factor Authentication

### Step 1: Sign In

1. Go to **https://mychart.unityhealth.to** or open the MyChart app.
2. Enter the MyChart username and password.
3. Select **Sign in**.

---

### Step 2: Select Verification Method

1. When prompted, select one of the following:
   - Email verification
   - Text message (SMS) verification
2. Select **Continue**.

---

### Step 3: Enter Verification Code

1. Retrieve the verification code sent by email or text message.
2. Enter the verification code.
3. Select **Verify**.

After verification:
- The sign-in process is complete.
- The user gains access to their MyChart account. [1](https://unityhealth.to/patients-and-visitors/mychart/how-to-use-mychart/)

---

## Procedure: Set Up a Passkey

### Step 1: Access Security Settings

1. Sign in to MyChart.
2. Navigate to **Account Settings**.
3. Select **Security Settings**.

---

### Step 2: Add a Passkey

1. Select **Set up passkey**.
2. Follow on-screen instructions for the device.
3. Confirm passkey creation using the device’s security method.

After setup:
- The passkey is linked to the device.
- The passkey can be used for future sign-ins. [1](https://unityhealth.to/patients-and-visitors/mychart/how-to-use-mychart/)

---

## Procedure: Sign In Using a Passkey

### Step 1: Open MyChart

1. Go to **https://mychart.unityhealth.to** or open the MyChart app.
2. Select **Sign in with passkey**, if available.

---

### Step 2: Authenticate

1. Complete device-based authentication:
   - Fingerprint, OR
   - Face recognition, OR
   - Device PIN

After authentication:
- The user is signed in without entering a password. [1](https://unityhealth.to/patients-and-visitors/mychart/how-to-use-mychart/)

---

## Important Notes

- Passkeys are tied to the device used during setup.
- Passkeys do not transfer automatically to new devices.
- Two-Factor Authentication may still be required in some situations.
