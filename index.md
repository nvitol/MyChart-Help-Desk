# MyChart: How to Sign Up Without an Activation Code

## Who Can Use This
- Patients **12 years and older**
- Patients who **do not have an activation link or code**

---

## How to Sign Up

### Step 1: Open MyChart
- Go to: https://mychart.unityhealth.to  
  **OR**
- Open the **MyChart app** from the App Store or Google Play

---

### Step 2: Start the Sign‑Up Process
1. Select **Sign up now**
2. Under **No Activation Code?**
3. Select **Sign up online**

---

### Step 3: Enter Required Information
- Complete all required fields (marked with an asterisk *)
- Select **I am not a robot**
- Select **Next**

---

### Step 4: Verify Contact Information
1. On the “Verify Contact Information” page, select **Send to my email**
2. Check your email for a **6‑digit verification code**
3. Enter the code
4. Select **Verify**

---

### Step 5: Create Login Credentials
Create:
- **Username**
  - Minimum 6 characters
  - Maximum 20 characters
- **Password**
  - Minimum 8 characters
  - Maximum 20 characters
  - Must include at least one letter and one number
  - Cannot match the username

Select **Next** when complete.

---

### Step 6: Set Communication Preferences
- Choose how you want to receive notifications
- Select **Sign in**

---

### Step 7: Accept Terms and Conditions
- Read the MyChart Terms and Conditions
- Select **Accept**

✅ Your MyChart account is now created.

---

## First‑Time Login: Two‑Step Verification
After your first login, you will be prompted to set up two‑step verification.

### Steps:
1. Select **Continue**
2. Add an email address or phone number
3. Select **Continue**
4. Enter the verification code sent to you
5. Select **Verify**

This helps protect your account.

---

## Common Issues
- **Did not receive verification code**
  - Check spam or junk email folder
  - Confirm email address is correct
- **Verification failed**
  - Re‑enter the code carefully
  - Request a new code if available
